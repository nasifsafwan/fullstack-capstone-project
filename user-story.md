# GiftLink User Story Template

## Template

**As a** `[type of user]`  
**I need** `[goal or capability]`  
**So that** `[reason or value]`

### Acceptance Criteria

- Given `[context]`
- When `[event]`
- Then `[expected outcome]`

## Example GiftLink Stories

1. As a giver, I need to create a listing for an unused item so that someone nearby can reuse it.
2. As a recycler, I need to search by category and location so that I can quickly find items that match my needs.
3. As a new user, I need to register securely so that I can manage my profile and saved preferences.
4. As a returning user, I need to log in with my credentials so that I can update my account information.
5. As a browser, I need to see the details of an item so that I know whether it is worth collecting.
6. As a user, I need to edit my profile so that my contact and pickup details remain accurate.
7. As an administrator, I need application builds to pass in CI so that deployment risks are reduced.
8. As a product owner, I need the backlog tracked in GitHub Issues so that progress is visible through the capstone.
